Researchers Uncover "Hallucination Squatting" Attack: Open-Source Models Match Top Models at Lower Cost

Security researchers recently disclosed a new AI supply-chain attack called "hallucination squatting." Large language models sometimes invent non-existent package names, function calls, and library imports when writing code. Attackers register these hallucinated names on PyPI or npm and inject malicious code. When a developer's CI pipeline pulls the dependency, the attack succeeds.

At the same time, Databricks released real-world coding-task tests showing that the Chinese open-source model GLM 5.2 matched Anthropic Opus 4.8 in task completion quality but cost only $1.28 per task, about 34% less than Opus's $1.94.

AI coding assistants' hallucinations can be weaponized into supply-chain attacks
